Resume Writing Checklist
✅Contact info: name, email, phone, LinkedIn, GitHub
✅Professional email address
✅Resume is 1-2 pages only
✅No spelling or grammar errors
✅Used bullet points throughout
✅Quantified achievements (e.g. increased sales by 30%)
✅Strong action verbs to start each bullet
✅Tailored resume for specific job
✅Skills section matches job description keywords
✅Education: GPA if above 7.5/10
✅Projects section with tech stack and impact
✅Saved as PDF with professional filename
⚠️No personal photo (unless required in your country)
⚠️No tables, graphics, or fancy formatting
⚠️No personal info like age, religion, marital status
🤖
ATS Optimization Guide
🔑 Use Keywords from Job Description
Copy exact phrases from the job posting. ATS matches exact keywords.
📐 Simple Formatting Only
Avoid tables, columns, text boxes. Use simple single-column layout.
📝 Standard Section Headings
Use: Work Experience, Education, Skills, Projects.
🔤 Spell Out Abbreviations
Write "Artificial Intelligence (AI)" not just "AI".
